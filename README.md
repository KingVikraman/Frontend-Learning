## What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages.

HTML defines the structure and content of a webpage by organizing text, images, links, forms, and other elements into a format that web browsers can understand and display.

HTML is not a programming language. Instead, it is a markup language that describes the structure of a document.

Example:

```html
<h1>Hello World</h1>
<p>This is my first webpage.</p>
```
